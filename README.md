# Corporate Training Program Management System

A Spring Boot REST API for managing corporate training programs, instructors, students, and courses.

## Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- MySQL
- RESTful APIs

## Features
- CRUD APIs for Instructor, Student, and Course
- Relational JPA mappings
- Pagination and filtering using Pageable
- Centralized exception handling
- Layered architecture (Controller, Service, Repository)

## Getting Started
1. Clone the repository
2. Configure `application.properties`
3. Run `SpringappApplication.java`
4. Test APIs using Postman or Swagger
